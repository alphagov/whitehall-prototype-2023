Acronyms

Definitions for acronyms that will appear if the user hovers the mouse over it.

Add each definition in its own paragraph at the end of the body copy.

      *[TLA]:Three-letter acronym
Addresses

Use the Insert menu to look up and add a saved Contact address.

Or you can add $A before and after an address to inset it.

      $A
      Address
      Here
      $A
Call to action

Add $CTA before and after a paragraph to highlight an action for the user.

      $CTA
      You must do this
      $CTA
Email links

Select an email address and use the Link button to make a Mailto link.

Or you can add < and > before and after an email address.

      <example@gov.uk>
Tables

Add | to separate data into columns and put each row on a new line.

      Name | Colour | Number
      -|-|-
      Apple | Green | 3
      Banana | Yellow | 7

Add - | - after the first row to mark it as column headers.

You can right align a column by putting : in the header marker.

      Name | Number
      -|-:
      Apple | 0.3
      Banana | 0.7
Bar charts

Add {barchart} at the end of numeric tables to display a simple bar chart.

      Name | Number | Number
      -|-|-
      Thing | 120 | 445
      Thing | 154 | 228
      {barchart}

If you include negative values, you must use {barchart negative}.

      {barchart negative}

Use {barchart compact} to save space by shrinking the chart.

      {barchart compact}

Use {barchart stacked} for stacked bars. The final column is used to display the total.

      {barchart stacked}
